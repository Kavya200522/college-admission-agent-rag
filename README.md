# 🎓 College Admission Agent using RAG (Retrieval-Augmented Generation)

This project is a smart AI-powered assistant that helps prospective students by providing accurate, real-time responses to their admission-related queries using IBM Watsonx and Retrieval-Augmented Generation (RAG).

---

## 🔗 Live Project Link
👉 [Open in IBM Cloud](https://dataplatform.cloud.ibm.com/wx/agents/d00a9124-39b9-42ef-ba0b-e9aae7be97b9?project_id=5ac14dad-3edc-4657-a1fb-ddeccd835968&context=wx)

---

## 💡 Features

- 🧠 Understands natural language queries from students
- 📚 Retrieves information from official institutional documents
- ✅ Provides real-time, trustworthy answers
- 📝 Answers questions about:
  - Admission policies
  - Eligibility criteria
  - Fee structure
  - Important deadlines
  - Course selection and FAQs
- 🚫 Reduces manual inquiry load
- 🌐 Enhances accessibility and transparency

## 🚀 Sample Questions to Ask
